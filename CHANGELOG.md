# Change Log

## [1.0.0] - 2026-04-14

- First marketplace release
- Added VS Code command support (`vsCommand`)
- Added configurable running colour (`runningColor`)
- Added `cwd` and `showOutput` options
- Tested and stabilised for public release

## [0.0.1] - 2026-03-02 (beta)

- Initial pre-release
- Configurable status bar buttons for shell commands and VS Code tasks
- Visual feedback with spinning icon and colour changes while running
- Configuration via `.vscode/command-buttons.json`
- File watcher for live config reload
- Kill running shell commands by clicking the button again
